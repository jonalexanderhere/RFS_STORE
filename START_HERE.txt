🎯 LANGKAH CEPAT (5 MENIT):

STEP 1: Buat File .env
━━━━━━━━━━━━━━━━━━━━━━
1. Buka file: CONFIG_ENV.txt
2. Copy semua isinya (Ctrl+A, Ctrl+C)
3. Buat file baru: .env (titik di depan!)
4. Paste isi tadi
5. Save

STEP 2: Setup Supabase
━━━━━━━━━━━━━━━━━━━━━━
1. Buka: https://supabase.com/dashboard
2. Sign Up / Login (gratis)
3. Buat project: "rfs-store-dev"
4. Tunggu 2 menit
5. Settings → API → Copy URL & anon key
6. Paste di file .env
7. SQL Editor → Run supabase-schema.sql

STEP 3: Jalankan
━━━━━━━━━━━━━━━━━━━━━━
1. Buka PowerShell di folder ini
2. Ketik: npm run dev
3. Buka: http://localhost:3000
4. DONE! ✅
