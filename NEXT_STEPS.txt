STEP 1: Setup Supabase Database
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Buka browser: https://supabase.com/dashboard

2. Login / Sign Up (gunakan GitHub atau Google - GRATIS)

3. Klik tombol "New Project"

4. Isi form:
   Name: rfs-store-dev
   Database Password: (klik "Generate a password")
   Region: Southeast Asia (Singapore)
   Plan: Free (sudah cukup)

5. Klik "Create new project"

6. TUNGGU 2 MENIT (project sedang dibuat)

7. Setelah selesai, klik Settings (gear icon) → API

8. Anda akan lihat:
   - Project URL (contoh: https://abcxyz123.supabase.co)
   - Project API keys → anon public (string panjang eyJ...)

9. COPY kedua nilai tersebut

STEP 2: Update File .env
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Buka file .env di folder RFS (file sudah ada!)

2. Cari baris ini:
   VITE_SUPABASE_URL=https://your-project-id.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.your-anon-key-here

3. GANTI dengan credentials Supabase Anda:
   VITE_SUPABASE_URL=https://abcxyz123.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...(paste key Anda)

4. SAVE file (Ctrl+S)

STEP 3: Setup Database Schema
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Di Supabase Dashboard, klik SQL Editor (icon </>) di sidebar

2. Klik "New query"

3. Di folder RFS, buka file: supabase-schema.sql

4. COPY SEMUA isi file tersebut (Ctrl+A, Ctrl+C)

5. Kembali ke Supabase SQL Editor

6. PASTE di editor (Ctrl+V)

7. Klik tombol "Run" atau tekan Ctrl+Enter

8. Tunggu 10-15 detik

9. Harus muncul: "Success. No rows returned" ✅

10. Verifikasi: Klik Table Editor → harus ada tables:
    - profiles
    - services  
    - orders
    - invoices
    - payment_proofs
    - notifications
    - audit_logs
